## Acceptance criteria

- [ ] Each Coup role has private goal text available to the assigned player.
- [ ] A shared Coup Rules Reference is available during setup or play.
- [ ] The reference states that every other player remains an opponent for Magic card text by default.
- [ ] The reference warns players not to prove hidden roles by showing the app screen unless a Coup rule reveals them.
- [ ] The reference includes Royal Guard, Inquisition, Green Eligibility, and Wasteland guidance.
