# Legacy Issues Archive

Status: archived

The previous issue tracker for this repo lived under `issues/`.

Those files were migrated to `.scratch/legacy-issues/archive/issues/` when the repo switched to the local markdown convention expected by the agent skills:

- Feature PRDs live at `.scratch/<feature-slug>/PRD.md`
- Implementation issues live at `.scratch/<feature-slug>/issues/<NN>-<slug>.md`

Use the archived files only as historical context. Do not create new active work in this legacy archive.
