# Coup Role Images

Place user-provided Coup role images here through the import script:

```sh
go run ./scripts/coup-images -source /path/to/role-images
```

Canonical embedded filenames use Coup role IDs:

- `1001.*` King
- `1002.*` Blue Knight
- `1003.*` Black Knight
- `1004.*` Red Knight
- `1005.*` Green Knight
- `1006.*` Wasteland Knight

Supported extensions are `.jpg`, `.jpeg`, `.png`, and `.webp`.
